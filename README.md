# Desafio técnico Full Stack

Corrigir problemas nos arquivos fornecidos conforme os requisitos especificados em cada desafio de frontend.

## Rodando Aplicação

com yarn:

```BASH
yarn install
yarn dev
```

com npm:

```BASH
npm install
npm run dev
```

## Rodando Testes

com yarn:

```BASH
yarn test
```

com npm:

```BASH
npm run test
```
